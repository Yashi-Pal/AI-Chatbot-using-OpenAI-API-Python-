# 🤖 AI Chatbot using OpenAI API (Python)

This project is a simple AI-powered chatbot built using Python and OpenAI API. It allows users to have a conversation with an AI model in real-time through the command line.

---

## 🚀 Features

- Interactive chatbot using OpenAI's language model
- Maintains conversation context
- Simple and easy-to-use CLI interface
- Built using Python

---

## 🛠️ Tech Stack

- Python
- OpenAI API
